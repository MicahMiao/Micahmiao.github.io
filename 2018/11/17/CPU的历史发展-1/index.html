基本工具类说明

主要对象

1. DataBaseConnect 类

功能：封装Mysql数据库各项操作

表1-1 DataBaseConnect成员变量表

变量名称 访问权限 数据类型 作用
connection private MySqlConnection 数据库连接对象
server private string 数据库IP地址信息
database private string 数据库名称
uid private string 数据库连接用户ID
password private string 数据库连接密码

表1-2 DataBaseConnect成员函数表

函数 功能 个人理解
DataBaseConnect() 从文件中读取相关数据库连接信息并初始化MySqlConnection对象 函数耦合严重
setCommand(String command) 封装MySqlCommand.ExecuteReader方法 函数未被引用
closeDataReader(MySqlDataReader dataReader) 关闭MySqlDataReader对象并关闭数据库链接 函数未被引用
ExecuteScalaer(String command) 封装 MySqlCommand.ExecuteScalar方法 函数未被引用
Execute(String command) 封装MySqlCommand.ExecuteNonQuery方法 连接状态未判断即关闭
Quarry(String command) 封装EMySqlCommand.ExecuteReader方法 设置一个MySqlDataReader对象来返回查询结果，查询失败返回null
Close() 关闭数据库连接 规范

3. SerialPortCommunication 类

功能：封装串口通信类

表3-1 SerialPortCommunication成员变量

变量名称 访问权限 数据类型 作用
_continue static bool 控制线程内用查询法读取数据
isReading static bool 判断读函数是否退出
_serialPort static SerialPort 实例化端口对象
serialPortData private static ArrayList 暂存从串口获取的数据
readThread Thread 读数据的线程
buffer byte[] 缓存 byte 数组
motionBuffer byte[] 缓存运动传感器数组
pressureBuffer byte[] 缓存压力传感器数组
commandBuffer byte[] 缓存命令的返回值
packageLength public const int 单个数据包的长度(244)
packageHead public const int 数据包头的位置？(0xFA)
motionPackageLength public const int 运动传感器数据包长度(221)
pressurePackageLength public const int 压力传感器数据包长度(205)
processedData ProcessedData 进程数据类对象
displayMessage Thread 测试缓冲区大小的线程
uniqueSerialPortCommunication private static SerialPortCommunication 静态 SerialPortCommunication
serialDataReceivedEventHandler SerialDataReceivedEventHandler 数据收到事件处理句柄(null)
sendCheckCommunicationStatus public static byte[] 发送开始发送数据命令字符串
sendStartSendData public static byte[] 发送停止发送命令字符串
sendStopSendData public static byte[] 发送停止发送命令字符串
receiveSuccessfulOperation public static byte[] 操作成功”字符串
receiveUnrecognizedCommand public static byte[] “命令无法识别”字符串
receiveOperationFailure public static byte[] “操作失败”字符串

表3-2 SerialPortCommunication成员函数

函数 功能 个人理解
SerialPortCommunication() 单例，私有构造函数 从文件中初始化SerialPort对象
GetInstance() 返回单例 无
DataReceivedHander(object sender, SerialDataReceivedEventArgs e) 开始 readThread 线程 DataReceive 事件所驱动的函数，用于开始接收数据并放入 serialPortData(ArrrayList) 中。
readSpeed() 测试读取速度 测试函数
IsOpen() 输出串口状态 测试函数
SerialPortText() 测试串口 测试函数
Open() 打开串口函数 Bug修复：重新创建读线程。一个线程运行完后不能再次start，要想再次start，只能new一个
Close() 关闭串口函数 无
getSerialportData() 获取收到的数据 获取直接从串口接收的数据
send(string command) 向串口发送字符串 已经废弃
sendCommand(Byte[] bytes) 向串口发送命令 已经废弃
Read() 读数据线程 无限循环读取缓冲区内的数据
readCommand() 读命令函数 已经废弃
attachDataReceivedEvent() 挂载“数据收到事件处理句柄”到“接收到数据”事件 向 _serialPort 的 DataReceived 事件添加函数
convertParity(String parity) 将 text 文件中取出的数据做类型转换 将int类型转换成party
convertStopBits(String stopBits) 将 text 文件中取出的数据做类型转换 将string类型转换成stopbits
convertHandshake(String handshake) 将 text 文件中取出的数据做类型转换 将string类型转换成handshake
SetPortName(string defaultPortName) Display Port values and prompt user to enter a port. 已经废弃
SetPortBaudRate(int defaultPortBaudRate) Display BaudRate values and prompt user to enter a value. 已经废弃
SetPortParity(Parity defaultPortParity) Display PortParity values and prompt user to enter a value. 已经废弃
SetPortDataBits(int defaultPortDataBits) Display DataBits values and prompt user to enter a value. 已经废弃
SetPortStopBits(StopBits defaultPortStopBits) Display StopBits values and prompt user to enter a value. 已经废弃
SetPortHandshake(Handshake defaultPortHandshake) Display Handshake values and prompt user to enter a value. 已经废弃
